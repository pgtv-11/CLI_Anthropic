# Written Supervisory Procedures (WSPs)

Versioned WSPs governing every supervisory function (FINRA 3110(b)). Each file is markdown, frontmatter-tagged with `policy_version` and `effective_from`. The Regulatory RAG service indexes this directory alongside the rulebook corpus so that "what does Avenue do for X?" can be answered with both the rule (FINRA/SEC) and Avenue's procedure.

Files live alongside their rule anchor, e.g. `wsp-3110-supervision.md`, `wsp-3310-aml.md`, `wsp-2210-comms.md`.

WSPs are reviewed annually and signed by the CCO; an unsigned/expired WSP is a finding.
